# 天谷Bot
## このBotについて
- このBotは「天谷bot作ろうぜwww」のような感じのノリで作ったものです。
- 指定した文字をランダムに送信する機能や、返信機能のオンオフ機能があります。

## インストール
[Node js V13.0+](https://nodejs.org/)をインストールする必要があります。
linuxの場合
``````bash
git clone https://github.com/MyokoJP/Amagai-Bot
cd Amagai-Bot
npm install
``````
windowsの場合
``````bash
ソースコードをダウンロードします。
Win ＋ Rでコマンドプロントを開きます。
ダウンロードした場所にcdで移動します。
npm install
``````

## 実行
linuxの場合
``````bash
cd Amagai-Bot
node index.js
``````
windowsの場合
``````bash
cd {ダウンロードした場所}
node index.js
``````
